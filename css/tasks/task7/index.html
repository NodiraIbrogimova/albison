Uzmedtezaurus

Kasallik nomlari semantik maydoni

CC ( Semantik soha ) — tibbiyot sohasi.

CM ( Semantik maydon ) — kasallik nomlari semantik maydoni.

CCM ( Semantik submaydon ) — kasallik tushunchasini umumlashtirib nomlovchi semantik sub maydon: kasal, dard, epidemiya, pandemiya, vabo, o'lat.

CMM ( Semantik mikromaydon ) — kasallikning aniq turini bildiruvchi semantik mikromaydon: jarohat, insult, bod, pilla.

CMM( Semantik mikromaydon ) — kasallik belgisini bildiruvchi semantik mikromaydon: asabiy, irsiy, majruh, shabko'r.
